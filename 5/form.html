POST Request

Form ini menggunakan method post untuk mengirimkan data untuk diproses oleh script di web server.

Request post mengirimkan data di dalam body request.

Biasanya digunakan apabila data yang dikirim jumlahnya besar, atau merupakan data binary atau tidak menginginkan data yang dikirim di-encode pada URL address bar.

Untuk proses upload dokumen (gambar atau file lainnya) hanya bisa dilakukan melalui request POST

GET Request

Form ini menggunakan method get untuk mengirimkan data untuk diproses oleh script di web server.

Request get mengirimkan data dalam bentuk yang di encode (url-encoded) pada URL address bar.

Request GET biasanya digunakan apabila request atau alamat URL request diinginkan untuk bisa dibookmark oleh user (misalnya halaman detail produk, berita, dll).

GET & POST Request

Form ini menggunakan method get dan post ke web server.

Form tetap menggunakan method post, hanya pada atribut action diisikan dengan variable get yang akan dikirim ke server.

Pada sisi server, variable data request dapat diakses melalui varible $_POST untuk data form, $_GET untuk data yang dkirim melalui url action dan $_REQUEST untuk mengakses semua variable request baik yang POST maupun yang GET.

Coba buka source code halaman ini dan ganti atribut action pada form ini menjadi request.php?mode=zodiak
